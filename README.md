# Bulk-Email-Sender
 This clean and modern Flask application provides a streamlined solution for bulk email automation. Featuring a   minimalist and intuitive interface, users can easily configure SMTP settings and dispatch messages to multiple   recipients. It is a polished, responsive, and well-designed tool that ensures efficient delivery with a professional touch.
──────────────────────────────────────────────────────────────────────────────────────────────────────────────────────╯
✦ How to Use the Bulk Email Sender

   1. Install Dependencies: Run pip install flask (as listed in requirements.txt).
   2. Launch the App: Run python app.py and open http://127.0.0.1:5000 in your browser.
   3. SMTP Setup:
      - Server/Port: e.g., smtp.gmail.com and 587.
      - Credentials: Enter your email and an App Password (required for Gmail/Outlook with 2FA).
   4. Prepare Recipients: Paste email addresses in the Recipients box, one per line.
   5. Compose & Send: Enter your subject and message, then click Send Emails. Monitor the status area for success or
      failure logs.
